## Проекты

### **Предсказание диабета с помощью логистической регрессии**
**Полноценный проект по предсказательному моделированию** — от исследовательского анализа данных до реализации логистической регрессии с нуля и сравнения с реализацией из Scikit-learn.

**Ключевые особенности:**
- Комплексный EDA с визуализацией медицинских признаков
- Ручная реализация алгоритма логистической регрессии
- Сравнение производительности: собственная vs. Scikit-learn реализация
- Анализ важности признаков для медицинских инсайтов

[**Репозиторий**](https://github.com/maqsshil/Diabetes-log-reg-prediction.git)

### **Классификация вина с оптимизацией гиперпараметров**

**Проект по многомодельной классификации** — сравнение RandomForest, логистической регрессии и метода опорных векторов с системным подбором гиперпараметров.

**Ключевые особенности:**
- Сравнительный анализ трёх алгоритмов классификации
- GridSearchCV для оптимального подбора гиперпараметров
- Визуализация метрик производительности моделей
- Анализ признаков для характеристик качества вина

[**Репозиторий**](https://github.com/maqsshil/wine-classification-hyperparameter-optimization.git)
